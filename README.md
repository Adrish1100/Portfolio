## 📖 Description

**Portfolio** is a simple and responsive personal portfolio website built using **HTML** and **CSS**. It showcases personal information, technical skills, projects, contact details, and other relevant information in a clean and beginner-friendly format.

The project serves as an online introduction and demonstrates fundamental web development concepts using semantic HTML and basic styling techniques.

---

## ✨ Features

* Personal introduction and profile section
* Dedicated skills showcase with structured tables
* Project portfolio displaying completed works
* Contact information with direct links (email, phone, GitHub)
* Simple navigation menu for quick section access
* Lightweight and easy-to-customize design

---

## 🚀 Prerequisites & Setup

### Requirements

* Any modern web browser (Chrome, Firefox, Edge, Safari)
* Optional: Visual Studio Code or any code editor

### Installation

1. Clone the repository:

```bash
git clone https://github.com/Adrish1100/Portfolio.git
```

2. Navigate to the project directory:

```bash
cd Portfolio
```

3. Open the main file in your browser:

```bash
start index.html
```

Or simply double-click the `index.html` file.

---

## 💻 Usage

Run the portfolio locally:

```bash
# Open the website
index.html
```

Make changes to the HTML or CSS and refresh the browser to see updates.

Example:

```html
<h1>About Me</h1>
<p>Welcome to my personal portfolio website.</p>
```
